Basic Statistical Concepts

Statistics: Interprets results into numerical values and denotes a particular measure to quantify data.

2 Kinds of Statistics

  • Descriptive: Focuses on summarizing and describing data without involving generalizations.
  • Inferential: Uses generalizations, predictions, and estimations based on a sample dataset.

Population vs. Sample

  • Population: The entire group or large group under study (e.g., MSU-CETD students).
  • Sample: A smaller subset of the population (e.g., Grade 9 Fisher of MSU-CETD).
Variables & Levels of Measurement

Variable: A characteristic or attribute that changes across different subjects or items.

  • Independent Variable: The variable that is manipulated or controlled.
  • Dependent Variable: The variable being tested and measured.

Data Types

  • Qualitative: Non-numerical values or categorical measures (e.g., labels, traits).
  • Quantitative: Numerical values representing how much or how many.
    • Discrete: Countable and finite values.
    • Continuous: Infinite values obtained through continuous measurements.

Levels of Measurement

  • Nominal: Level used for identifying or classifying objects (e.g., gender).
  • Ordinal: Levels involving rankings or ordered sequences (e.g., top students, teacher rankings).
  • Interval: Numerical scale where the difference between measures is meaningful, but lacks a true zero.
  • Ratio: Level containing a true zero, used to measure height, area, and volume.
Measures of Central Tendency

These measures determine the central point or typical value within a given data sequence.

  • Mean (Average): Calculated by taking the sum of all measurements divided by the total number of items (n). Symbolized by x̄.
  • Median (Middle): The exact middle position of a sorted set of measurements.
  • Mode: The value or category that occurs most frequently.
    • Unimodal: Exactly 1 mode is present.
    • Multimodal: More than 1 mode is present.
    • No Mode: No numbers are repeated in the sequence.

Formula: μ = Σfx / N

Quartiles, Deciles, & Percentiles

Quantiles split a sorted distribution into equal fractional divisions.

1. Quartiles (Four Equal Parts)

Formula for position: iQ_k = (k/4)(n + 1)

Example Sequence: 24, 31, 33, 37, 37, 39, 40, 45, 46, 46, 48, 49, 52, 55, 60 (n = 15)

  • Position of Q1 = (1/4)(16) = 4th position → Q1 = 37
  • Position of Q2 = (2/4)(16) = 8th position → Q2 = 45
  • Position of Q3 = (3/4)(16) = 12th position → Q3 = 49

2. Deciles (Ten Equal Parts)

Formula for position: iD_k = (k/10)(n + 1)

Example for D1 position: (1/10)(16) = 1.6 position (Interpolation needed: 24 + 0.6 * (31 - 24) = 28.2)

3. Percentiles (One Hundred Equal Parts)

Formula for position: iP_k = (k/100)(n + 1) or calculated via sample indices.

Median & Quantiles for Grouped Data

To determine locations of partition values within frequency distribution tables, specialized interpolation formulas are applied.

Grouped Median Formula

M_m = L_m + [((N/2) - F_m) / f_m] * c

  • L_m: Lower boundary of the median class
  • N: Total cumulative frequency
  • F_m: Cumulative frequency below the median class
  • f_m: Frequency of the median class
  • c: Size of the class interval

Grouped Quartile Formula

Q_k = LB + [((kN/4) - CF_b) / f_Qk] * i

Variance & Standard Deviation

Measures how spread out data values are from the calculated center mean.

Population Equations

  • Variance (σ²) = Σf(x - μ)² / N
  • Standard Deviation (σ) = √[Variance]

Sample Equations

  • Sample Variance (s²) = Σf(x - μ)² / (N - 1)
  • Sample Standard Deviation (s) = √[Sample Variance]

Steps to Solve: Find Midpoint (x) → Calculate Mean (μ) → Find Deviation (x - μ) → Square it (x - μ)² → Multiply by frequency f(x - μ)² → Sum items and divide.

Dispersion Coefficients & Skewness

Quartile Coefficient of Dispersion

Formula: V_q = [(Q3 - Q1) / (Q3 + Q1)] * 100%

Coefficient of Dispersion

Formula: V = (σ / μ) * 100%

Skewness (Distribution Irregularity)

Formula: S_k = (μ - mo) / σ

  • Symmetrical: Distribution is perfectly centered (μ = median = mode).
  • Right-Skewed (Positive): Distribution tails to the right side (μ > median > mode).
  • Left-Skewed (Negative): Distribution tails to the left side (μ < median < mode).
Pearson's R & T-Test Foundations

Measures the linear relationship strength and directional path between two variable sets (x & y).

  • Positive Correlation (+): As variable x increases, variable y simultaneously increases.
  • Negative Correlation (-): As variable x increases, variable y decreases.
  • Range Limits: Standardized score limits remain strictly bounded between -1.00 and +1.00.

T-Test Analysis

A statistical testing method utilized to verify if the computed numeric differences are statistically valid or simply due to random variation.

  • Null Hypothesis (H_0): Standard benchmark hypothesis that agrees directly with the default baseline claim.
  • Alternative Hypothesis (H_1): Direct structural contradiction to the default baseline claim.
